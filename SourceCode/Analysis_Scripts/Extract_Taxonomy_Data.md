# Extract_Taxonomy_Data.R Documentation

## Overview
Processes taxonomy data for ecological plots, generating summary statistics and normalized metrics for each test site.

## Main Functionality
- Loads taxonomy data from Excel files.
- Prepares and summarizes data for each test site.
- Outputs normalized summary statistics.

## Dependencies
- R packages: vegan, readxl, writexl, openxlsx, dplyr, NbClust, dbscan, ggplot2, tidyr, pracma

## Usage
Set input paths and run the script to generate taxonomy summaries and normalized metrics for each test site.
